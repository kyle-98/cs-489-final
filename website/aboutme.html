I'm a Computer Science student here at Virginia Wesleyan University. I have created this project for my senior capstone research class. I am currently very interested in not only programming, but meteorlogy as well as space exploration.

My passion for meteorlogy led me to this project idea. I thought this would be a good project to not only develop skills in Javascript, but to be able to get a taste of being a full-stack developer working on both the backend and the front-end of a website. While the main research for the project was data analysis and visualization of data for facitiliation to the public, I enjoy researching and learning about meteorlogy as a hobby. These two things combined together to end up with this final project.
