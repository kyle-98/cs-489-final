About the Project

This project contains analysis and visualization on data provided by NOAA on past tropical cyclones that have existed in the Atlantic basin since the year 2000. The website contains visualizations from graphs and charts to different types of maps including heatmaps and an interactive map. There is detail about each storm and the dates on each occured. Analysis of the data gives users a range of options to view and understand the datasets.

The research for this project was data analysis and how to analyze data with different programming languages. The languages used to analyze data in this project are Javascript, including AJAX & JQuery, and Python. Data analysis is very important on many different levels.
